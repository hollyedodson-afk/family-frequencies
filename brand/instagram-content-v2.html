Highlight covers

Navy circle + gold icon for each: About (swell), What's On (calendar), The Disco (ticket), Good to Know (info). Grab the swell from the logo files; the rest match the brand's icon style.

How to roll it out

Post the three brand posts first to set up the account, then run the campaign: poster announcement → what-to-expect → reminder the day before. The ticker, spinning stamp and bobbing logo are live animations — screen-record any frame for a Reel.
